# Code of Conduct

This project follows the Contributor Covenant spirit: be respectful, constructive, and professional.

## Expected Behavior

- Be respectful of differing viewpoints and experience levels.
- Give actionable, technical feedback.
- Focus criticism on code and design decisions, not people.
- Avoid harassment, insults, or personal attacks.

## Unacceptable Behavior

- Harassment or intimidation
- Trolling or inflammatory comments
- Publishing others' private information without permission
- Any conduct that would be inappropriate in a professional engineering setting

## Reporting

Please report serious misconduct privately to the project maintainer through GitHub contact channels instead of opening a public issue.

## Enforcement

Project maintainers may edit, remove, or reject contributions and community interactions that are not aligned with this code of conduct.
